The Archive

Welcome to The Archive! This is a historical timeline of AKI rom hack releases and versions through the years.

If you want to try one of the historical versions of a rom hack you see here out, just click the link on the Version column and you can download a patch file. You can then use that patch file and the correct rom with your favorite rom patching software or Rom Patcher JS. If you need to find the correct rom to patch, read the README file for that patch, found by clicking the file icon on the end of the row.
